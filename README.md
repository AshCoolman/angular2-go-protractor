# Angular 2 Go !

Simple starter example that gets you going with Angular 2 in minutes.

## Pre-Requisites

1. Must install pre-requisites

	```bash
	npm install -g typescript live-server
	```

## Getting Started

1. Clone this repo

1. Run `npm install`

1. Run the TypeScript compiler and watch for changes `npm run tsc`

1. Open 2nd terminal and launch the app in the browser `npm start`

## Notes

This sample intentionally uses precise versions of Angular 2 and SystemJS so new versions do not break it. I will update these as Angular moves out of Alpha.
